¿Qué es y para qué usar JQuery?

Todos los navegadores web modernos incluyen de forma nativa información para procesar muchas funciones básicas de JavaScript, que permiten funcionalidades elementales. JQuery es una librería que contiene otras funcionalidades adicionales, que nos permiten agregarle a las páginas mucha más interacción y efectos.

Como usar JQuery

El primer requisito para usar JQuery es agregar la librería, que podemos cargar desde su principal repositorio o desde otros CDN de internet.
En JQuery se pueden usar como selectores elementos HTML como a (enlaces) h1, h2, etc.
También clases usando (.nombre) e identificadores (#nombre).

Sintaxis

Elementos

Es posible también usar otros eventos como dblclick, hover, keypress, keyup, focus y otros. Es posible encadenar múltiples métodos en una sola línea de código. Esto permite realizar múltiples operaciones en los elementos seleccionados de forma más concisa. Los métodos se encadenan uno tras otro, separados por puntos (.).

Sintaxis

Elementos

Selector

✔Es una expresión que indica qué elementos HTML se van a seleccionar. Puede ser una etiqueta HTML, una clase, un ID, un atributo, una relación entre elementos, entre otros. Se utiliza $() para crear un objeto jQuery y pasarle el selector como argumento.

Metodo

✔ Representa una acción que se realizará en los elementos seleccionados. Los métodos en jQuery se encadenan después del selector y se invocan mediante un punto (.) seguido del nombre del método. Los métodos pueden incluir manipulación del DOM, eventos, animaciones, entre otros.

Paramentro

✔ Algunos métodos de jQuery aceptan parámetros adicionales que se utilizan para personalizar su comportamiento. Los parámetros se colocan entre paréntesis después del nombre del método y se separan por comas si hay más de uno.

FUNCIONES

✔ Algunos métodos de jQuery aceptan una función de devolución de llamada como parámetro. Esta función se ejecutará después de que se complete la acción del método.

Animaciones

En los primeros ejemplos usamos varios métodos de efectos de JQuery, para mover, ocultar y mostrar elementos, usando animaciones.


BLOQUE DE PRUEBA

Todos los efectos anteriores se consiguen usando los siguientes métodos:


Hide (Ocultar Bloque)

Se utiliza para ocultar elementos HTML en una página web. Cuando se llama a la función hide(), los elementos seleccionados se ocultan y ya no se muestran en la página.

Show (Mostrar Bloque)

En jQuery se utiliza para mostrar elementos HTML que han sido ocultados previamente. Cuando se llama a la función show(), los elementos seleccionados se vuelven a mostrar en la página.

Toggle (Ocultar / Mostrar)

En jQuery se utiliza para alternar entre mostrar y ocultar elementos HTML en una página web. Si un elemento está visible, la función toggle() lo ocultará, y si está oculto, la función toggle() lo mostrará.

Animate ( Cambiar Tamaño y Mover)

En jQuery se utiliza para crear animaciones personalizadas en elementos HTML. Permite cambiar gradualmente los valores de uno o varios atributos CSS de un elemento a lo largo de un período de tiempo determinado.

fade Toggle (Ocultar con FadeOut)

Se utiliza para alternar entre el estado de visibilidad de un elemento con una animación de desvanecimiento (fade). Si el elemento está visible, se desvanecerá gradualmente y se ocultará. Si está oculto, se desvanecerá gradualmente y se mostrará.

fadeIn (Mostrar con FadeIn)

Se utiliza para mostrar elementos HTML gradualmente con una animación suave. Con la función fadeIn(), los elementos seleccionados se muestran de manera gradual, desde estar ocultos hasta estar completamente visibles.

Efectos

Los efectos en jQuery son métodos y funcionalidades que permiten agregar animaciones y transiciones a los elementos HTML seleccionados. Estos efectos añaden dinamismo e interactividad a las páginas web, mejorando la experiencia del usuario.
